# git-commit-history

## Description

An example app displaying the commit history for this repository.

- Built With:
    * JavaScript
    * Node.js
    * React
    * Bootstrap


![](./public/screenshot.png)


## Installation

~~~
npm install
~~~

## Testing

~~~
npm start
~~~

* Access the app by navigating to http://localhost:3001/
